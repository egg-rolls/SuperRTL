## 使用流程

```bash
# 1. 安装
pip install superrtl

# 2. 初始化项目
superrtl init --name my_project --top top_module

# 3. 编写代码 (AI 辅助)
superrtl skills show fsm          # 获取设计模式
superrtl skills show spi          # 获取 SPI 设计

# 4. 编译验证
superrtl build                    # 自动依赖排序编译
superrtl test                     # 运行所有测试

# 5. 查看波形
superrtl waveform view simulation.vcd
```

## 支持的设计模式

### 基础模块
- FSM (有限状态机)
- FIFO (同步/异步)
- 计数器 / 移位寄存器
- 译码器 / 多路选择器

### 通信接口
- SPI (主从)
- I2C (主机)
- UART (收发器)
- AXI-Lite (从机)

### 数字信号处理
- FIR 滤波器
- PWM 生成器
- CRC 校验

### 可靠性设计
- CDC (跨时钟域)
- 复位同步器
- 按键消抖
- 边沿检测
- 仲裁器
